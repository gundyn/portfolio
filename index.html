Stage 01 — Trigger

GitHub Actions Workflow

Listens for every push to main and kicks off the full build-test-deploy sequence automatically.

GitHub Actions YAML
deploy.yml View 
on:
  push:
    branches: [main]
  workflow_dispatch:

jobs:
  validate-html:
  lint-css:
  check-links:
  deploy:
    needs: [validate-html,
            lint-css,
            check-links]
Stage 02 — Quality

Automated Checks

Three parallel jobs gate every deploy — HTML validation, CSS linting, and a broken-link scan. All must pass before deploy runs.

CI Checks Stylelint Lychee
HTML validation html-validate
CSS linting stylelint
Broken links lychee
Stage 03 — Delivery

Deploy to Pages + Cloudflare

Once all checks pass, automated deployment to GitHub Pages with Cloudflare handling DNS, caching, and edge delivery — zero manual steps.
